# دليل سريع لتشغيل البوت 24/7

## الطريقة الأسهل والأسرع

### 1. استخدام `run_bot_service.py` (مُوصى بها)

```bash
python run_bot_service.py
```

هذا الأمر سيشغّل البوت مع إعادة التشغيل التلقائي في حالة الفشل.

### 2. استخدام `start_bot.bat` (للمستخدمين على Windows)

- انقر نقراً مزدوجاً على `start_bot.bat`
- أو افتح Terminal وشغّل:
```bash
start_bot.bat
```

### 3. استخدام `start_bot_forever.bat` (للخلفية)

- انقر نقراً مزدوجاً على `start_bot_forever.bat`
- سيشغّل البوت في نافذة منفصلة في الخلفية

---

## المتطلبات

1. **Python 3.8+** مثبت على النظام
2. **ملف `.env`** يحتوي على `BOT_TOKEN`
3. **جميع المكتبات المطلوبة** مثبتة (`pip install -r requirements.txt`)

---

## إيقاف البوت

- اضغط `Ctrl+C` في Terminal
- أو أغلق النافذة
- أو استخدم Task Manager لإنهاء العملية

---

## مراقبة البوت

- **اللوغات**: تحقق من `bot.log` و `bot_service.log`
- **الحالة**: أرسل `/start` للبوت للتحقق من أنه يعمل
